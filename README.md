# BSC / EVM Swap Bot

## Задача

Сделай бота, который:

- Принимает адрес токена
- Через Dexscreener находит пулы
- Выбирает пул с самой большой ликвидностью - а может с самой лучшей ценой? ну ок.
- Делает swap (покупка токена) - а на что покупает?

> Поддерживаем только V2 / V3 / V4
  V4 (Infinity): Пока не индексируется отдельно в Dexscreener, нужно мониторить.
> Сеть только BSC

## Ожидаемое

- запуск через CLI (или простая команда)
- логирование: что нашли на dexscreener, какой пул выбрали, tx hash
- `.env.example` и норм README
